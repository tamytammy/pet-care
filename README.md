#  寵物健康紀錄App

本專案使用Vue3去製作一個結合行事曆，紀錄寵物健康醫療等資訊之應用程式。

##  預期專案功能

- 儲存寵物資訊
- 個別寵物可新增健康資料
- 回診、吃藥通知提醒
- 每日輸入資訊以圖表呈現
- 可將資料輸出成PDF, excel等檔案作其他使用(獸醫)
